## Media keys

| Shortcut            | Description        |
| ------------------- | ------------------ |
| `Super + =`         | Increase volume    |
| `Super + -`         | Decrease volume    |
| `Super + ,`         | Play previous song |
| `Super + .`         | Play next song     |
| `Super + Backspace` | Stop music         |
| `Super + Return`    | Play music         |


## Unity

| Shortcut            | Description         |
| ------------------- | ------------------- |
| `Alt + Space`       | Show unity launcher |
